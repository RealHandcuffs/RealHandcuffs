1.  For development, always use standard edition: Install Standard Edition and then copy everything from MO to Fallout 4/Data.
2.  Before updating, remove all files that belong to compatibility plugins.
3.  Make sure all modified scripts are compiled and RealHandcuffs.esp is saved. Use BAE to extract RealHandcuffs - Main.ba2. Do not overwrite changed files!
4.  Use pack_assets.bat to rebuild RealHandcuffs - Main.ba2.
5.  Update the 0_Common folder:
    - Data/AAF
    - Data/MCM/Config/RealHandcuffs
    - Data/Scripts/Source/User/RealHandcuffs
    - Data/RealHandcuffs - Main.ba2
    - Data/RealHandcuffs - Textures.ba2
6.  Build the 1_Standard folder:
    - Data/RealHandcuffs.esp
7.  Open RealHandcuffs.esp with FO4Edit and wait for background loader to finish.
    - Remove the override of 'Container': 001E7DC9 'QAMiscComponentChest'
    - Remove the overide of 'FormID List': 0017c667 'lootItemsUncommon'
    - Remove the overide of 'FormID List': 00246EE7 'CA_JunkItems'
    - Remove all overrides in 'Leveled Item'
    - Remove the override of 'Misc. Item': 00059B3F 'Handcuffs'
    - Expanding and scrolling through all categories, there should be no edit to existing forms
    - Set the value of 'Global': XY00089D 'RH_IsLiteEdition' to 1
    - Set the value of 'Global': XY0008DC 'RH_ChanceNoneRestraints' to 100
    - Set the value of 'Global': XY0008DD 'RH_ChanceNoneKeys' to 100
    - Set the value of 'Global': XY000937 'RH_ChanceNoneRemoteTrigger' to 100
    - Close FO4Edit and save RealHandcuffs.psc
8.  Build the 2_Lite folder:
    - Data/RealHandcuffs.esp
